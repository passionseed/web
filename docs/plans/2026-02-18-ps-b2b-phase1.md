# PassionSeed B2B Phase 1 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a working Phase 1 backend pipeline for B2B lead discovery, scoring, outreach drafting, and manual CRM feedback learning.

**Architecture:** Implement a typed domain module under `lib/ps-b2b` and expose one orchestration API route. Keep discovery manual-list based for MVP and design module interfaces so automated providers can be added in Phase 2.

**Tech Stack:** Next.js App Router, TypeScript, Zod, AI SDK (optional outreach assist), Jest.
