## Assessment Prompt

**Question:** เขียน สมมติฐาน ที่พิสูจน์ได้ ใช้ format นี้:

พวกเราเชื่อว่า [กลุ่มคน] ต้องการ [สิ่งที่จะสร้าง] เพราะ [หลักฐานจาก Phase 1]

พวกเราจะรู้ว่าถูก ถ้า [ผลลัพธ์ที่วัดได้]  
พวกเราจะรู้ว่าผิด ถ้า [ผลลัพธ์ที่บอกว่าผิด]

ต้องเจาะจง พิสูจน์ได้ และอิงจากหลักฐานใน Problem Proof Pack

**Min Words:** 50

**Example Answer:**

พวกเราเชื่อว่าผู้สูงอายุที่เป็นเบาหวานในกรุงเทพต้องการตัวเตือนยาผ่าน LINE เพราะ 7/8 คนที่สัมภาษณ์ลืมกินยาอย่างน้อยสัปดาห์ละครั้ง และทุกคนใช้ LINE อยู่แล้ว พวกเราจะรู้ว่าถูก ถ้า 3/5 คนที่ทดสอบบอกว่าจะใช้ พวกเราจะรู้ว่าผิด ถ้าน้อยกว่า 2/5 คนกรอกแบบฟอร์มลงทะเบียน

## Grading Criteria

### Pass (ผ่าน)
- สมมติฐาน เจาะจงและพิสูจน์ได้
- อิงจากหลักฐานจาก Phase 1
- มีเงื่อนไข "ถูก" และ "ผิด" ที่วัดได้

### Needs Revision (ต้องแก้ไข)
- สมมติฐาน คลุมเครือ
- ไม่อ้างอิงหลักฐานจาก Phase 1
- ไม่มีเงื่อนไขที่ชัดเจน

### Auto-Fail (ตก)
- ไม่อ้างอิงหลักฐานจาก Phase 1 เลย
- สมมติฐาน ไม่พิสูจน์ได้
