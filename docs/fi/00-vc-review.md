## Remaining Concerns

### 1. Revenue Model Still Needs Validation
**Issue**: Multiple models mentioned, need to pick ONE and validate.

**Recommendation**: 
- Focus on B2C parent-pays model for initial traction
- Add B2B school licensing for scale (higher LTV, lower churn)
- Test pricing: ฿500-2,000/month for premium features

### 2. Unit Economics Not Yet Validated
**Issue**: CAC $3.00 and LTV $45.00 are estimates.

**Recommendation**: 
- Run paid acquisition tests in Thailand + Vietnam
- Track actual conversion rates from free → paid
- Measure real LTV from paying customers

### 3. Content Quality at Scale
**Issue**: AI-generated modules need to be high-quality to retain users.

**Recommendation**:
- Start with 5-10 high-quality expert interviews
- Measure student completion rates and satisfaction
- Build feedback loop to improve AI output

### 4. Regulatory Considerations
**Issue**: Student data privacy across SEA countries.

**Recommendation**:
- Consult with legal on PDPA (Thailand), PDPA (Singapore), etc.
- Build privacy-first from day one
- Consider local data residency requirements
