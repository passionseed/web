# style guideline

## Typography

- **Primary Font**: Space Mono (monospace) used throughout the entire website
- Font weights: 400 (regular) and 700 (bold)

## Color Palette

- **Primary Theme**: Horizon-inspired gradient transitioning from darker purple galaxy to reddish-orange dawn
- **Purple Shades**:

- Purple-950 (`#3b0764`) - darkest
- Purple-900 (`#4c1d95`)
- Violet-900 (`#5b21b6`)
- Violet-800 (`#6d28d9`)

- **Red/Orange Shades**:

- Red-600 (`#dc2626`)
- Red-700 (`#b91c1c`)
- Orange-600 (`#ea580c`)
- Yellow-400 (`#fbbf24`) for accents

## Layout Structure

- **Header**: Sticky navigation with logo, main nav links, and user authentication
- **Main Content**: Gradient backgrounds transitioning through the color palette
- **Footer**: Features animated lava effect with reddish color scheme

## Component Design

- **Cards**: Semi-transparent backgrounds with backdrop blur effects
- **Buttons**: Consistent styling with hover states matching the color theme
- **Progress Bars**: Color-coded based on skill/interest levels
- **Badges**: Category-specific colors (yellow for Inspire, orange for Build, red for Scale)

## Animation & Effects

- **Lava Footer**: Canvas-based animation with flowing lava particles in red/orange tones
- **Gradient Backgrounds**: Smooth transitions between purple galaxy and orange dawn themes
- **Interactive Elements**: Hover effects and smooth transitions

## Workshop Categories

- **PassionSeed (Inspire)**: Yellow/amber color scheme (`#d97706`)
- **Passion Growth (Build)**: Orange color scheme (`#ea580c`)
- **Passion Scale (Connect)**: Red color scheme (`#dc2626`)

## User Interface Patterns

- **Authentication**: Discord-branded login with consistent styling
- **Data Visualization**: Progress bars, emotion indicators, and statistical displays
- **Responsive Design**: Mobile-first approach with grid layouts
- **Dark Theme**: Primary theme with light text on dark backgrounds

The overall aesthetic creates a cosmic, inspiring feel that transitions from the mystery of a purple galaxy to the warmth and energy of an orange dawn, perfectly embodying the "passion seed" concept of growth and transformation.
